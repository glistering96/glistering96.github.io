Backend for Production AI

AI 서비스를 운영 가능한 백엔드로 만들어요.

AI/ML 모델과 복잡한 물류 도메인 로직을 API, queue, data contract, runtime, observability로 연결해 운영 가능한 서비스로 만듭니다.

10분 → 2~3분 자동차선 적부 추론 시간
900초 → 1초 복화 매칭 응답 성능
Scale-out Queue + Redis 상태 일관성
3초 내 B2C 매칭 API 응답 제약 대응

문제의 본질은 모델 성능이 아니라 운영 가능한 형태로 바꿨을 때입니다. 도메인 제약을 데이터 계약으로 정합시키고, 긴 작업은 callback, 동시성은 queue, 관측은 로그·메트릭으로 설계해 실제 서비스로 운용했습니다.
